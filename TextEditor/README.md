Text Editor
===========

A text editor made using Tkinter.

Features  
1. Save files
2. Open files
3. Copy/Cut selected text
4. Paste text from clipboard
5. CLear text
6. Word count
7. Commom keybindings like  
   * Ctrl-c - Copy
   * Ctrl-v - Paste
   * Ctrl-x - Cut
   * Ctrl-z - Undo
   * Ctrl-a - Select all
   * Ctrl-s - Save
   * Ctrl-o - Open
